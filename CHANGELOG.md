## [1.0.0] - 2025-02-15

### Added
- Initial release
- CKEditor 4 Livewire component
- CKEditor 5 Livewire component
- TipTap Livewire component
- `@livewireEditorAssets` Blade directive
- Configuration file for customizing editor defaults
- CSS styling for all editors
- JavaScript helper functions
- Comprehensive documentation
- Example usage files
- Support for Livewire 3.x
- Support for Laravel 10.x and 11.x
- Dark mode support for TipTap editor
- Custom configuration per editor instance
- Event dispatching on content updates

### Features
- Multiple editor choices (CKEditor 4, CKEditor 5, TipTap)
- Easy asset management with single Blade directive
- Highly configurable components
- Two-way data binding with Livewire
- CDN and local asset support
- Publishable views, config, and assets
- Clean and organized package structure
- MIT licensed
